# 🏋️ Gerenciador de Treinos

Aplicação de terminal para gerenciar treinos de academia, desenvolvida em Python com orientação a objetos.

## Funcionalidades

- Criar treinos (A, B, C...)
- Adicionar exercícios com séries e repetições
- Listar todos os treinos e exercícios

## Tecnologias

- Python 3.14
- Orientação a Objetos
- Sem frameworks ou bibliotecas externas

## Como rodar

```bash
python main.py
```

## Aprendizados

- Classes e objetos
- Encapsulamento
- Métodos especiais (`__init__`, `__str__`)
- Organização de código em módulos
